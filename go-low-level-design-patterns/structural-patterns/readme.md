## 2. Structural Patterns

### How classes and objects are composed to form larger structures.

```sh
| Pattern       | Used For                                     | Real-world Example                                   |
| ------------- | -------------------------------------------- | ---------------------------------------------------- |
| **Adapter**   | Integrate incompatible APIs                  | Integrating multiple payment APIs (Razorpay, PayPal) |
| **Decorator** | Add behavior without changing code           | Logging, rate limiting, metrics wrapping middleware  |
| **Proxy**     | Control access, caching, or lazy loading     | Redis caching proxy, API gateway                     |
| **Facade**    | Simplify complex subsystems                  | Service SDKs, database utility packages              |
| **Composite** | Treat individual and group objects uniformly | Menu/category trees, org hierarchies                 |

```
